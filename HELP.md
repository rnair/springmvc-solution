# Spring MVC demo solution

### Frameworks used

* Spring Boot
* Spring web starter (Spring MVC)
* Apache Tiles3
* Spring data JPA (Hibernate)
* H2 Inmemory DB
* Jquery
* JqueryUI
* Jquery DataTable

### Build and Deploy

* mvn clean install
* java -jar /target/springmvc-solution-0.0.1-SNAPSHOT.war
